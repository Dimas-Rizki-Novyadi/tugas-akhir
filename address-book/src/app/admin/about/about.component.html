ADDRESS BOOK

Address book is a simple, easy and fast website-based application. You can collect other people's address data easily in a fast process. This web application can be used by people of all ages, from young to old. All address data that is considered important will be stored safely. The purpose of this web application is to fulfill the assignments for the Advanced Web Technology course at my place of study, namely Ahmad Dahlan University.
